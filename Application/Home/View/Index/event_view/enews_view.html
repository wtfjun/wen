论坛简介 2014-11-25 17:12

 文化科技融合创新已成为当前全球文化产业发展的重要驱动力。2011年，深圳大学文化产业研究院申报的科研课题——“文化与科技融合创新的内在机理与战略路径研究”获得国家社会科学基金重大项目立项。自2012年起，深圳大学和南山区人民政府共同发起举办了文化科技创新论坛（CTIS），旨在搭建一个中国大陆、港澳台地区和发达国家的文化产业政产学研交流平台。2012、2013文化科技创新论坛在深圳成功举办后，不仅在文化产业研究领域广受好评，而且引起社会各界广泛关注，被《人民日报》、《中国文化报》、《中国社会科学报》、《南方日报》、《深圳特区报》、广东卫视、深圳卫视等各大媒体报道，同时也被凤凰、腾讯、新浪、搜狐、网易等众多主流网站转载、传播。基于论坛研讨成果的《文化科技蓝皮书——文化科技创新发展报告》也每年由社会科学文献出版社出版发布。

    为深入研讨当前文化科技创新发展的新趋势，我们于2014年11月29-12月1日举办“2014文化科技创新论坛（CTIS2014）”。本次论坛主题为“移动互联背景下的文化科技融合”。我们致力于将CTIS打造成文化科技创新领域的高层次、小规模、常设性的国际论坛。
